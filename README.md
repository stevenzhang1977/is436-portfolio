# Lab4 Portfolio

This project is about developing a professional personal portfolio website using the Bootstrap framework. This portfolio showcases my skillset, resume, projects, and learning outcomes from IS 436 - Structured Systems Analysis and Design while applying strong UI/UX design principles.

The website includes pages for:
- Home — Introduction & call-to-action
- About Me — Bio, background, and learning journey
- Portfolio / Projects — Showcases 3 projects with tech stack
- Skills & Resume — Technical skills, tools, and expertise
- Contact — Contact form and my contact information.

This project also incorporated the use of wireframing to design a blueprint of the website before doing any coding.
